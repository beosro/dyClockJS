dyClockJS

dyClockJS is a small plugin to create digital and analog clock for your blog and website.

Download GitHub

Created by: Yusuf Shakeel

Click here for the documentation.

Prerequisite

jQuery v1.12+ is required for dyClockJS.

Setup

<!DOCTYPE html>
<html>
  <head>
    <title>Index Page</title>
    <link type="text/css" rel="stylesheet" href="css/default.css">

    <!-- dyClockJS style -->
    <link rel="stylesheet" href="dyClockJS/css/dyclock.min.css">

  </head>
  <body>

    <!-- put your content here -->

    <div class="some-class">
      <div id="digital-clock" class="dyclock-container"></div>
    </div>

    <div class="some-class">
      <div id="analog-clock" class="dyclock-container"></div>
    </div>

    <!-- javascript -->
    <script src="js/jquery.min.js"></script>
    <script src="dyClockJS/js/dyclock.min.js"></script>
    <script>
      // digital clock
      var digitalClock = new dyClock($("#digital-clock"));
      digitalClock.start();

      // analog clock
      var analogClock = new dyClock($("#analog-clock"), {
        clock : "analog"
      });
      analogClock.start();
    </script>

  </body>
</html>
